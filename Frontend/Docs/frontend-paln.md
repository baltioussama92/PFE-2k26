Create a modern frontend for a house renting platform similar to Airbnb.

Tech stack:

* React
* Tailwind CSS
* Responsive design (mobile, tablet, desktop)
* Clean and minimal UI

Color palette:

* Primary: #0F172A (dark navy)
* Accent: #F59E0B (orange for buttons)
* Background: #F8F7F4 (light neutral)

Do NOT include pricing pages or testimonials.

Create the following pages and components:

1. Home Page

* Full screen hero section with a background image of a modern house
* Large heading like "Find Your Perfect Stay"
* Search bar with fields:

  * Location
  * Check-in date
  * Check-out date
  * Guests
  * Search button
* Featured property cards
* Categories section (Apartments, Villas, Cabins, Beach Houses)
* Footer with links

2. Search Results Page

* Grid layout of property cards
* Filters sidebar (price range, property type, amenities)
* Sort options (price, rating, newest)

3. Property Details Page

* Image gallery
* Property title and location
* Description
* Amenities list
* Host information
* Reviews section
* Booking sidebar with calendar and "Reserve" button

4. Login Page

* Email and password fields
* Login button
* Link to register page

5. Register Page

* Name
* Email
* Password
* Confirm password

6. User Profile Page

* User information
* My bookings
* Saved properties

7. Add Property Page (Host dashboard)

* Form fields:

  * Property title
  * Description
  * Location
  * Price
  * Amenities
  * Image upload

8. Navigation Bar

* Logo
* Home
* Explore
* Become a Host
* Profile menu

Use reusable React components:

* Navbar
* PropertyCard
* SearchBar
* Footer
* FilterPanel

Make the design modern, similar to Airbnb, with rounded cards, shadows, and clean spacing.
