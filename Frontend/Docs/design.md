# House Renting Platform – Design System

## 1. Design Philosophy

The interface follows a **luxury real-estate aesthetic** inspired by modern property platforms.
The design focuses on **simplicity, elegance, and visual comfort**, allowing property images to be the main focus.

Key design principles:

* Minimal color palette (4 colors only)
* Large high-quality property images
* Rounded cards and containers
* Clean typography
* Spacious layout
* Soft shadows
* Premium and calm visual atmosphere

The experience should feel like browsing **luxury vacation homes**.

---

# 2. Color Palette

The platform uses **only four main colors** to maintain a consistent and premium look.

## Beige – Main Background

HEX: `#F5EFE6`

Usage:

* Main page background
* Section backgrounds
* Soft UI areas

Purpose:
Creates a warm and welcoming feeling similar to natural home interiors.

---

## White – Content Surfaces

HEX: `#FFFFFF`

Usage:

* Cards
* Forms
* Property containers
* Image frames

Purpose:
Keeps the interface clean and improves readability.

---

## Black – Text and Contrast

HEX: `#111111`

Usage:

* Headings
* Paragraph text
* Icons
* Footer
* Navigation text

Purpose:
Provides strong contrast and improves readability.

---

## Gold – Accent Color

HEX: `#C6A969`

Usage:

* Primary buttons (Book / Reserve)
* Highlights
* Hover states
* Important icons

Purpose:
Adds a luxury feeling and draws attention to actions.

---

# 3. Color Usage Ratio

To maintain visual balance:

* 50% Beige
* 25% White
* 15% Black
* 10% Gold

Gold should be used **sparingly** so important elements stand out.

---

# 5. Layout Structure

## Navigation Bar

Top centered navigation with rounded container.

Menu items:

* Home
* About
* Explore
* Contact

Right side elements:

* Language selector
* Phone contact

---

## Hero Section

Large background image of a luxury house.

Elements included:

* Main title (large serif heading)
* Property tags (e.g. Mountain Retreat, Cozy Cottage)
* Booking form
* Call-to-action button

Booking fields:

* Check-in date
* Check-out date
* Phone number
* Book a House button

---

## About House Section

Section presenting the property concept.

Content includes:

* Image gallery cards
* Property description snippets
* Gallery button
* Customer review preview

Cards should have:

* Rounded corners
* Soft shadows
* Large images

---

## Property Information Section

Displays important details about the property.

Example information:

* Description
* Amenities list
* Number of rooms
* Year built

Example layout:

Rooms: 6
Year: 2023

---

# 9. Image Style

Images should follow these guidelines:

* High resolution
* Warm lighting
* Interior or exterior property photography
* Natural lifestyle feel
* Rounded image corners

Property photos should be the **visual focus of the interface**.

---

# 10. Overall Experience

The platform should feel:

* Calm
* Elegant
* Minimal
* Comfortable
* Premium

The design should give users the feeling of **discovering beautiful homes and planning a relaxing stay**.
